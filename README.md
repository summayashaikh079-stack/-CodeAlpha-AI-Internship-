# 🤖 Python AI Projects Collection

A collection of beginner-friendly Artificial Intelligence projects developed using **Python**. These projects demonstrate practical implementations of AI concepts, Natural Language Processing (NLP), music generation, and language translation through simple graphical user interfaces built with **Tkinter**.

---

## 📌 Projects Included

### 1. 💬 AI FAQ Chatbot
An intelligent FAQ chatbot that matches user questions with predefined FAQs using **TF-IDF Vectorization** and **Cosine Similarity**.

#### Features
- Interactive GUI using Tkinter
- Intelligent question matching
- Instant responses
- Beginner-friendly NLP implementation

#### Technologies
- Python
- Tkinter
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

---

### 2. 🎵 AI Music Generator
A simple AI-inspired music generator that creates random melodies and exports them as a MIDI file.

#### Features
- Random melody generation
- MIDI file export
- Easy-to-use graphical interface
- Tempo control

#### Technologies
- Python
- Tkinter
- Music21
- Random Module

---

### 3. 🌍 Language Translator
A desktop language translator that translates text between different languages using Google Translate.

#### Features
- Translate between multiple languages
- Simple GUI
- Fast translation
- Supports language codes

#### Technologies
- Python
- Tkinter
- Deep Translator
- Google Translator API

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Python-AI-Projects.git
```

Move into the project folder

```bash
cd Python-AI-Projects
```

Install dependencies

```bash
pip install -r requirements.txt
```

Or install them manually

```bash
pip install scikit-learn music21 deep-translator
```

# ▶️ Running the Projects

### Run Chatbot


### Run Music Generator
